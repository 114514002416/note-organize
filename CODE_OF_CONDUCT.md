# 社区行为准则

## 我们的承诺

作为 Note Organize 项目的贡献者和维护者，我们致力于为所有人提供一个尊重、包容的社区。

我们承诺所有参与者，无论其年龄、身体特征、种族、国籍、性别、性别认同、性取向、宗教、经验水平或任何其他身份特征，都能获得尊重和友好的对待。

## 我们的标准

### 鼓励的行为

- 使用欢迎和包容的语言
- 对不同的观点和经历表示尊重
- 优雅地接受建设性批评
- 专注于对社区最有利的事情
- 向其他社区成员表示同情

### 不可接受的行为

- 使用性化语言或意象
- 个人攻击、侮辱或贬低评论
- 骚扰或骚扰
- 出版他人的私人信息，如物理地址或电子邮件地址，未经明确许可
- 其他可能被合理认为是专业环境中不适当的行为

## 执行

项目维护者负责澄清可接受行为的标准，并应对任何不可接受行为采取适当的纠正措施。

对于骚扰、虐待或其他不可接受行为的报告，请通过以下方式与项目负责人联系：

📧 **邮件**：your.email@example.com

所有投诉都将被审查和调查，并将以维护人员认为适当的方式进行回应。

## 范围

本行为准则适用于项目内所有空间，也适用于代表项目的个人。代表项目的方式包括使用项目电子邮件地址、通过官方社交媒体账户或在在线或离线活动中担任指定代表。

## 后果

行为不当可能导致以下后果：

1. **警告** - 维护者可能会发出警告
2. **评论删除** - 可能会删除违反行为准则的评论
3. **问题或 PR 关闭** - 可能会关闭不遵守指南的 Issue 或 PR
4. **贡献者禁止** - 在严重或重复违反行为准则的情况下，可能会禁止参与者

## 报告准则

如果你认为有人正在骚扰或违反本行为准则，请立即报告。

报告应包含：
- 你的联系信息
- 违规者的身份（如果已知）
- 发生的情况的描述
- 发生时间和地点
- 任何其他相关信息

## 归属
